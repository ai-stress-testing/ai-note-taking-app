# ai-note-taking-app (NeuroVim)

Local-first, education-focused note/study editor built on TanStack Start.
No backend — state persists to `localStorage`; AI features require a local
Ollama server.

See [CLAUDE.md](./CLAUDE.md) for architecture, conventions, and the
spec-driven development workflow used in this repo.

## Getting started

```
bun install
bun run dev
```
